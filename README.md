## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/nobruf/shadcn-landing-page.git
```

2. Go into project

```bash
cd shadcn-landing-page
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
